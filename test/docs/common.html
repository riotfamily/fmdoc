The originating request URI.

includeUri

The path within the application. If the request is included via a RequestDispatcher, the variable contains the URI of the include, otherwise it's the same as documentUri.

Includes the given path using a RequestDispatcher. The argument may also be a sequence of paths, in which case multiple includes are performed.
Parameters:
path (optional, default is "")

encodeLinks

encodeLinks(html)
Adds the contextPath and sessionId to all links found in the given HTML if necessary.

Example:

${encodeLinks('Hello <a href="/world.html">World</a>')} 
==> Hello <a href="/context/world.html;jsessionid=...">World</a>
Parameters:
html (required)

href (required)

partition

partition(collection, property)
Partitions a collection by inspecting the specified property of the contained items. If the property value is different than the previous one, a new group is created and added to the returned sequence. Each group consists of a hash with a 'title' and an 'items' property.
Parameters:
collection (required) property (required)

fileExtension

fileExtension(filename, validExtension=[], defaultExtension="")
Returns the extension of the given file-name. If the validExtensions parameter is specified, the defaultExtension will be returned if the actual extension is invalid. ${fileExtension('foo.html')} => html ${fileExtension('foo.bar', ['gif', 'jpg'], 'unknown')} => unknown
Parameters:
filename (required) validExtension (optional, default is []) defaultExtension (optional, default is "")

formatMillis

formatMillis(millis)
Returns a formatted string using the pattern hh:mm:ss. The hours are omitted if they are zero, the minutes are padded with a '0' character if they are less than 10.
Parameters:
millis (required)
